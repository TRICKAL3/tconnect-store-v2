## Use a Different Project Name

Since `tconnect-store-v2` is already used for the frontend, use a different name for the backend.

## Step-by-Step:

### 1. Create New Project
1. Go to Vercel Dashboard
2. Click **"Add New"** â†’ **"Project"**
3. Import from GitHub
4. Select `tconnect-store-v2` repository

### 2. Configure with Different Name
1. **Project Name:** Use one of these:
   - `tconnect-backend`
   - `tconnect-api`
   - `tconnect-store-backend`
   - `backend-tconnect`
   - Or any name you like!

2. **Framework:** `Other`

3. **Root Directory:** `backend` (type exactly)

4. **Build Command:** Leave EMPTY

5. **Output Directory:** Leave EMPTY

6. **Install Command:** `npm install`

7. Click **"Deploy"**
